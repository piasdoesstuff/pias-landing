# PIAS Landing Page — V1 Wireframe (Single Page)

## Layout principles
- Strong whitespace
- Minimal blocks
- Clear visual rhythm
- One accent color + monochrome base

## Block map (top to bottom)

### Block A — Hero (70–85vh)
- Left/top-aligned title stack:
  - PIAS
  - Personal Intelligence Augmentation System
  - One-line value proposition
- Right/behind layer: subtle interactive ASCII field
- CTA row (2 buttons)

### Block B — Capability Grid
- 2 columns desktop, 1 column mobile
- 5 compact cards:
  - Strategy
  - Execution
  - Automation
  - Iteration
  - Security-minded

### Block C — Collaboration Flow
- Horizontal stepper on desktop / vertical on mobile
- 4 steps with short one-liners
- Minimal iconography (optional)

### Block D — Proof Strip
- Tight timeline/list of project outcomes
- Can be a vertical “log style” list with dates optional

### Block E — Principles
- 4 bullet principles with generous spacing
- Low visual weight but high clarity

### Block F — Final CTA
- Short closing statement
- One primary CTA button
- Optional secondary contact link

## Spacing system
- Section vertical spacing: 96px desktop / 64px tablet / 48px mobile
- Content max width: 1120px
- Internal card padding: 24px

## Empty space guidance
- Keep top hero breathing room large
- Avoid filling every row with elements
- Maintain asymmetric negative space in hero and principles sections
